This page allows you to add information regarding the bio-source of an particular sample. upon entering the information in this page, you will be asked to enter other aspects of sample like growth conditions, harvest and treatment conditions applied to that particular sample. Field by Field description is given below

  1. Organ Specification - Name or description of specific part of organ if any used for the study
  2. Cell Type - Specific plant cells used after dissection or cell sorting or any other method. This naming should be used based on Plant Ontology Consortium (POC).
  3. Sub Cellular Location - This is about sub cellular locations from cells are collected for experiment. Naming should be maintained based on Gene Ontology Cellular Component
  4. BioSource Amount - This is reference to the mass (mg fresh weight or mg dry weight), number of cells or other measurable quantification information
